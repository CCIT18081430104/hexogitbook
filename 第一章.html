第一章：本地搭建Hexo博客

  • 第一步：安装与配置必备软件和环境

    Node.js (选择好路径后，一直下一步即可)

    npm (npm跟随Node.js安装包，可以不用重复安装)

    git(官方网站) 或者 git淘宝镜像 （两种相同。前者国外网站下载速度慢，后者国内下载速度更快)

    安装完成后 使用 WIN+R 输入cmd 打开系统 dos 界面 (如图1-1)

    【图1-1】

    依次输入

node -v

【图1-2】

npm -v

【图1-3】

git

【图1-4】

按步骤依次输入并回车后 得到以上版本号或者代码即为 安装成功！

hexo安装

【可选项】把npm源改为淘宝镜像源可大大加快npm下载速度（如果用此方法以后使用npm时改为cnpm）

在dos命令簿输入

npm config set registry http://registry.npm.taobao.org/

完成即可

安装hexo

npm install hexo-cil -g

若已安装淘宝镜像源则为

cnpm install hexo-cli -g

等待安装，出现版本号即安装成功

我们还可以输入

hexo -v

来检查是否安装成功 如【图1-5】则安装成功

【图1-5】

  • 第二步：搭建本地博客

    1. 博客初始化

    首先在你的硬盘内创建一个名为 blog 的 空 文件夹

    在刚刚的DOS命令簿内输入

    md blog
cd blog

    md为新建文件夹 cd为进入blog文件夹

    此步骤也可以在硬盘根目录下 右击新建文件夹 用此方法记住利用cd命令进入blog文件夹内

    【图1-6】

    初始化

    输入代码

    hexo init

    等待下载完成即可进入下一步

    2. 大功告成！本地预览博客

    启动！

    输入代码

    hexo s

    如【图1-7】

    【图1-7】

    可以看到 提示 开始处理， Hexo 已经运行在 http://localhost:4000的地址上，按Ctrl+C 停止处理，按Ctrl+C后 这个地址就没有作用了。

    这时，博客已经生成了!
    打开浏览器 输入 http://localhost:4000 即可看到你的博客初始页

【图1-8】
本地博客搭建已完成，第一章结束
